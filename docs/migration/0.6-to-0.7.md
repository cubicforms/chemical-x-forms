# Migrating from 0.6.x to 0.7.x

One breaking change that touches application code, plus a type
tightening.

## `handleSubmit` now returns a handler, not a Promise

**0.6:**

```ts
const onSubmit = () =>
  handleSubmit(async (values) => {
    await api.post('/signup', values)
  })
```

**0.7:**

```ts
const onSubmit = handleSubmit(async (values) => {
  await api.post('/signup', values)
})
```

```vue
<template>
  <form @submit.prevent="onSubmit">...</form>
</template>
```

The template doesn't change — `onSubmit` changes from a thunk to
a handler. Bind it directly to `@submit.prevent` or call it
imperatively (`await onSubmit()`). The returned handler accepts
an `Event` directly, so `preventDefault` composes cleanly.

## Type surface tightened

No runtime change. The types around `UseAbstractFormReturnType`
were narrowed. If you were casting to `any` around `register` or
`handleSubmit`, try removing the cast — the compiler usually
agrees now.
